The user wants to share their personal skill with the Formidable OSS GitHub organization.

First, the user must be made aware that the skill will be public through the [`vercel-labs/skills` cli](https://github.com/vercel-labs/skills)

Only then, skills can start being copied to the discovery `skills` folder. should be created in the skills folder, under a new folder based on the github ID or username or whatever the user wants to choose to represent himself.
