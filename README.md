# Formidable Builders shareable skills

### Contribution

1. fork & branch using your name as a prefix, i.e. `gabimoncha/feat-*`
2. create a folder with your username, i.e. `skills/gabimoncha/`
3. use the `/formidable-skill` and say what skill you want to share and where
4. edit the copied skill if needed
5. commit and open a PR
6. don't send slop
